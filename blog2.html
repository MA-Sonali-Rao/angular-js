Blog 2: Setting Up the Environment

To start building with AngularJS, you don't need a heavy installation. You just need to link the library.

Step 1: The Library Link

You can include AngularJS in your HTML file using a Content Delivery Network (CDN) link:

<script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.8.2/angular.min.js"></script>

Step 2: The ng-app Directive

This is the most important step. You must tell HTML where the Angular application starts by using ng-app.
