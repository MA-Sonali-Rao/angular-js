Blog 7: Expressions & Filters

Expressions are used to bind application data to HTML. They are written inside double braces: {{ expression }}.

What are Filters?

Filters can be added to expressions to transform the data. For example:

  • {{ name | uppercase }} - Changes text to all caps.
  • {{ price | currency }} - Formats a number as money.
  • {{ date | date:'fullDate' }} - Formats a date nicely.
