Blog 4: Understanding Directives

Directives are markers on a DOM element that tell AngularJS to attach a special behavior to that element.

Common Directives:

  • ng-app: Initializes an AngularJS application.
  • ng-init: Defines initial values for an application.
  • ng-model: Binds the value of HTML controls (input, select, textarea) to application data.
