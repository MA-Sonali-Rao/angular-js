Blog 3: What is MVC?

AngularJS is based on the MVC (Model-View-Controller) design pattern.

Model: This is the data. (e.g., A user's name).

View: This is what the user sees. (The HTML page).

Controller: This is the brain. It connects the data to the view.

By separating these three, your code stays organized and professional—which is exactly what big companies look for!
